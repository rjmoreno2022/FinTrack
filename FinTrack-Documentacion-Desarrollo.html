Introduccion y Vision

Proposito de FinTrack

FinTrack es una aplicacion web de finanzas personales disenada para personas que viven en economias con multiples monedas, particularmente en Venezuela donde coexisten el Bolivar (VES) y el Dolar Estadounidense (USD) como medios de intercambio cotidianos.

La aplicacion permite a los usuarios gestionar sus finanzas personales de forma integral: cuentas bancarias, efectivo, criptomonedas, transacciones diarias, deudas, presupuestos y metas de ahorro. Todo en una interfaz moderna, rapida y segura.

Problema que Resuelve

En Venezuela, la gestion financiera personal enfrenta desafios unicos:

  • Bimonetarismo: El usuario recibe ingresos en USD pero paga gastos en VES, necesitando conversiones constantes.
  • Multiple fuentes de fondos: Cuentas bancarias internacionales (Cayman), exchanges crypto (Binance), efectivo en diferentes ubicaciones, y cuentas bancarias locales (BVC, Mercantil).
  • Falta de herramientas: Las apps existentes no manejan bien la dualidad USD/VES ni las tasas de cambio variables.
  • Presupuestos invisibles: Es dificil rastrear en que se gasta el dinero cuando las transacciones ocurren en diferentes plataformas y monedas.
  • Deudas informales: Creditos tipo "Cashea", deudas con familiares, pagos diferidos del odontologo.

Objetivos del Proyecto

  1. Centralizar toda la informacion financiera del usuario en un solo lugar.
  2. Proveer conversion automatica entre USD y VES usando tasas actualizadas.
  3. Permitir categorizacion completa de ingresos y gastos.
  4. Facilitar la creacion y seguimiento de presupuestos mensuales.
  5. Visualizar el progreso de metas de ahorro (ej: Fondo de Emergencia 5000 USD).
  6. Gestionar deudas activas y programas de pago.
  7. Generar reportes claros con graficos e inteligencia financiera.

Usuarios Objetivo

El usuario principal es una persona residente en Venezuela que recibe ingresos en dolares (remoto, freelance o negocio) y necesita gestionar sus finanzas en ambas monedas. El perfil incluye:

  • Profesionales remotos que reciben salario en USD.
  • Emprendedores que manejan multiples cuentas.
  • Usuarios de Binance P2P para conversion VES/USD.
  • Personas con deudas informales que necesitan seguimiento.
  • Ahorristas con metas financieras definidas.
💡

Decision clave: Aunque el usuario vive en Venezuela, la aplicacion esta construida de forma generica para que pueda usarse en cualquier pais con economia dual o multi-moneda. Las tasas de cambio son configurables y extensibles.

Por que estas tecnologias

.NET 8 LTS

Version de soporte largo con mejoras significativas de rendimiento (hasta 20% mas rapido que .NET 6), minimal APIs mejoradas, y native AOT. El soporte LTS garantiza actualizaciones de seguridad hasta noviembre 2026, dando estabilidad al proyecto.

Blazor Server (con migracion a Auto)

Se elige Blazor Server para el MVP porque:

  • Menor complejidad: no requiere API separada ni WebAssembly.
  • Payload inicial pequeno: la app se renderiza en el servidor.
  • Acceso directo a la base de datos sin necesidad de API REST.
  • En fases posteriores se puede migrar a Blazor Auto o WASM con APIs.

MediatR + CQRS

CQRS separa operaciones de lectura (Queries) de escritura (Commands), permitiendo optimizar cada una por separado. MediatR facilita la comunicacion entre capas sin dependencias directas, promoviendo el desacoplamiento y haciendo el codigo mas testeable.

MudBlazor

En vez de crear componentes desde cero, MudBlazor provee un set completo de componentes de alta calidad (tablas, formularios, dialogs, tabs, charts) con tema oscuro integrado, responsividad y accesibilidad. Acelera el desarrollo del frontend semanas.

FluentValidation sobre Data Annotations

FluentValidation permite validaciones complejas, condicionales y reutilizables fuera de las entidades. Es mas expresivo que Data Annotations y facilita las pruebas unitarias de reglas de validacion.

Nota sobre la version de MudBlazor: Asegurate de usar MudBlazor 6.x o superior compatible con .NET 8. Si usas .NET 8, verifica la compatibilidad en la documentacion oficial antes de instalar.

Arquitectura de Software

Clean Architecture

FinTrack sigue los principios de Clean Architecture (tambien conocida como Onion Architecture), organizando el codigo en capas concentricas donde las dependencias apuntan siempre hacia el centro: el dominio.

Domain-Driven Design (DDD)

El modelo de dominio es el corazon de la aplicacion. Se aplica DDD en los siguientes aspectos:

  • Entidades: Objetos con identidad unica que persisten en el tiempo (Account, Transaction, Debt, Goal).
  • Value Objects: Objetos sin identidad propia, definidos por sus atributos (Money, Currency).
  • Aggregates: Cluster de entidades y value objects tratados como una unidad. Transaction es parte del aggregate Account.
  • Aggregate Roots: Entidad raiz que controla el acceso al aggregate (Account, Debt, Goal).
  • Domain Events: Eventos que representan algo que ocurrio en el dominio (TransactionCreated, GoalReached).
  • Repositories: Abstraccion para persistir y recuperar aggregates.

Patrones de Diseno Aplicados

Repository Pattern

Abstrae el acceso a datos detras de interfaces. Cada aggregate tiene su propio repositorio. La capa de aplicacion depende de IRepository<T>, no de Entity Framework directamente.

Unit of Work

Garantiza que todas las operaciones de un caso de uso se completen como una unidad atomica. El DbContext de EF Core actua como Unit of Work; SaveChangesAsync() persiste todo o nada.

CQRS (Command Query Responsibility Segregation)

Separacion de comandos (escritura) y queries (lectura):

  • Commands: CreateAccountCommand, UpdateTransactionCommand - modifican estado.
  • Queries: GetAccountByIdQuery, GetTransactionsListQuery - solo leen datos.
  • MediatR enruta cada request al handler correspondiente.

Dependency Injection

Todas las dependencias se inyectan a traves de interfaces. El contenedor DI de ASP.NET Core resuelve las implementaciones en tiempo de ejecucion. Esto facilita el testing con mocks.

Result Pattern

En lugar de lanzar excepciones para flujos de negocio esperados, los handlers retornan un objeto Result<T> que indica exito o fallo con mensaje descriptivo.

Specification Pattern

Encapsula criterios de filtrado complejos en objetos reutilizables. Por ejemplo: ActiveAccountsSpec, TransactionsByDateRangeSpec.

Principios SOLID Aplicados

PrincipioAplicacion en FinTrack
S - Single Responsibility Cada clase tiene una sola razon para cambiar. Los Commands solo crean, los Queries solo leen, los Validators solo validan.
O - Open/Closed Nuevas validaciones se agregan como nuevas clases sin modificar las existentes. Nuevas categorias via seed data.
L - Liskov Substitution IRepository<T> puede sustituirse por EFRepository o InMemoryRepository para tests sin afectar la aplicacion.
I - Interface Segregation IReadableRepository<T> e IWritableRepository<T> separados. Las queries no ven metodos de escritura.

Value Object: Money

Money representa un valor monetario con su moneda. Es inmutable y se compara por valor, no por referencia.

Domain/ValueObjects/Money.cs
public class Money : ValueObject
{
    public decimal Amount { get; }
    public Currency Currency { get; }

    public Money(decimal amount, Currency currency)
    {
        if (amount < 0)
            throw new DomainException("El monto no puede ser negativo");

        Amount = amount;
        Currency = currency;
    }

    public Money Add(Money other)
    {
        if (Currency != other.Currency)
            throw new DomainException("No se pueden sumar montos de diferentes monedas");

        return new Money(Amount + other.Amount, Currency);
    }

    public Money Subtract(Money other)
    {
        if (Currency != other.Currency)
            throw new DomainException("No se pueden restar montos de diferentes monedas");

        return new Money(Amount - other.Amount, Currency);
    }

    public Money ConvertTo(Currency targetCurrency, decimal rate)
    {
        if (Currency == targetCurrency)
            return this;

        var converted = Amount * rate;
        return new Money(converted, targetCurrency);
    }

    public override string ToString() => $"{Currency} {Amount:N2}";

    protected override IEnumerable<object> GetEqualityComponents()
    {
        yield return Amount;
        yield return Currency;
    }
}

Entidad: Account

Account es un Aggregate Root. Controla sus transacciones y garantiza la consistencia del balance.

Domain/Entities/Account.cs
public class Account : Entity, IAggregateRoot
{
    public string Name { get; private set; }
    public AccountType Type { get; private set; }
    public Currency Currency { get; private set; }
    public decimal Balance { get; private set; }
    public string? Description { get; private set; }
    public bool IsActive { get; private set; }
    public ICollection<Transaction> Transactions { get; private set; }

    private Account() { } // EF Core requiere constructor vacio

    public Account(string name, AccountType type, Currency currency,
                 string? description = null)
    {
        if (string.IsNullOrWhiteSpace(name))
            throw new DomainException("El nombre de la cuenta es requerido");

        Name = name;
        Type = type;
        Currency = currency;
        Balance = 0;
        Description = description;
        IsActive = true;
        Transactions = new List<Transaction>();
    }

    public Transaction AddTransaction(TransactionType type, decimal amount,
        string description, DateTime date, Guid? categoryId = null,
        string? tags = null, bool isRecurring = false)
    {
        if (!IsActive)
            throw new DomainException("No se pueden agregar transacciones a una cuenta inactiva");

        var transaction = new Transaction(Id, type, amount, Currency,
            description, date, categoryId, tags, isRecurring);

        UpdateBalance(type, amount);
        Transactions.Add(transaction);

        AddDomainEvent(new TransactionCreatedEvent(transaction.Id, Id, amount, type));

        return transaction;
    }

    public void UpdateInfo(string name, string? description)
    {
        if (string.IsNullOrWhiteSpace(name))
            throw new DomainException("El nombre es requerido");

        Name = name;
        Description = description;
        UpdatedAt = DateTime.UtcNow;
    }

    public void Deactivate()
    {
        IsActive = false;
        UpdatedAt = DateTime.UtcNow;
    }

    private void UpdateBalance(TransactionType type, decimal amount)
    {
        Balance = type switch
        {
            TransactionType.Income    => Balance + amount,
            TransactionType.Expense   => Balance - amount,
            TransactionType.Transfer  => Balance - amount, // La cuenta destino recibe
            _ => throw new DomainException("Tipo de transaccion no valido")
        };
    }
}

    {

            throw new DomainException("El acreedor es requerido");

    {
        if (amount <= 0)
            throw new DomainException("El pago debe ser mayor a cero");
        if (amount > RemainingAmount)
            throw new DomainException("El pago no puede exceder el monto pendiente");
        if (Status != DebtStatus.Active)
            throw new DomainException("No se pueden registrar pagos en una deuda no activa");

        var payment = new DebtPayment(Id, amount, date, note);
        Payments.Add(payment);
        RemainingAmount -= amount;

        if (RemainingAmount == 0)
            Status = DebtStatus.Paid;

        UpdatedAt = DateTime.UtcNow;
    }

    public void MarkAsDefaulted()
    {
        if (Status == DebtStatus.Paid)
            throw new DomainException("Una deuda pagada no puede marcarse como incumplida");
        Status = DebtStatus.Defaulted;
        UpdatedAt = DateTime.UtcNow;
    }

    public decimal ProgressPercentage =>
        OriginalAmount > 0 ? ((OriginalAmount - RemainingAmount) / OriginalAmount) * 100 : 0;
}

        GoalPriority priority = GoalPriority.Medium)
    {
        if (string.IsNullOrWhiteSpace(name))
            throw new DomainException("El nombre de la meta es requerido");
        if (targetAmount <= 0)
            throw new DomainException("El monto objetivo debe ser mayor a cero");

        Name = name;
        TargetAmount = targetAmount;
        CurrentAmount = 0;
        Currency = currency;
        TargetDate = targetDate;
        Description = description;
        Priority = priority;
        Status = GoalStatus.Active;
    }

    public void Contribute(decimal amount)
    {
        if (amount <= 0)
            throw new DomainException("La contribucion debe ser positiva");
        if (Status != GoalStatus.Active)
            throw new DomainException("Solo metas activas pueden recibir contribuciones");

        CurrentAmount += amount;

        if (CurrentAmount >= TargetAmount)
        {
            CurrentAmount = TargetAmount;
            Status = GoalStatus.Completed;
            AddDomainEvent(new GoalReachedEvent(Id, Name));
        }

        UpdatedAt = DateTime.UtcNow;
    }

    public void Withdraw(decimal amount)
    {
        if (amount <= 0 || amount > CurrentAmount)
            throw new DomainException("Monto de retiro invalido");

        CurrentAmount -= amount;

        if (Status == GoalStatus.Completed && CurrentAmount < TargetAmount)
            Status = GoalStatus.Active;

        UpdatedAt = DateTime.UtcNow;
    }

    public decimal ProgressPercentage =>
        TargetAmount > 0 ? (CurrentAmount / TargetAmount) * 100 : 0;

    public decimal RemainingAmount => TargetAmount - CurrentAmount;
}

Behavior de Validacion Pipeline

public class ValidationBehavior<TRequest, TResponse>
    : IPipelineBehavior<TRequest, TResponse>
    where TRequest : IRequest<TResponse>
    where TResponse : Result, new()
{
    private readonly IEnumerable<IValidator<TRequest>> _validators;

    public ValidationBehavior(IEnumerable<IValidator<TRequest>> validators)
    {
        _validators = validators;
    }

    public async Task<TResponse> Handle(TRequest request,
        RequestHandlerDelegate<TResponse> next, CancellationToken ct)
    {
        if (!_validators.Any()) return await next();

        var context = new ValidationContext<TRequest>(request);
        var results = await Task.WhenAll(
            _validators.Select(v => v.ValidateAsync(context, ct)));

        var failures = results.SelectMany(r => r.Errors)
            .Where(f => f != null).ToList();

        if (failures.Any())
        {
            var errors = failures.Select(f => f.ErrorMessage).ToArray();
            return (TResponse)(object)Result.Fail(errors);
        }

        return await next();
    }

public class ErrorHandlingMiddleware
{
    private readonly RequestDelegate _next;
    private readonly ILogger<ErrorHandlingMiddleware> _logger;

    public ErrorHandlingMiddleware(RequestDelegate next,
        ILogger<ErrorHandlingMiddleware> logger)
    {
        _next = next;
        _logger = logger;
    }

    public async Task InvokeAsync(HttpContext context)
    {
        try
        {
            await _next(context);
        }
        catch (DomainException ex)
        {
            _logger.LogWarning(ex, "Domain exception: {Message}", ex.Message);
            await HandleDomainExceptionAsync(context, ex);
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Unhandled exception occurred");
            await HandleExceptionAsync(context, ex);
        }
    }

    private static async Task HandleDomainExceptionAsync(HttpContext context, DomainException ex)
    {
        context.Response.StatusCode = StatusCodes.Status400BadRequest;
        context.Response.ContentType = "application/json";
        var response = new { error = ex.Message, type = "DomainError" };
        await context.Response.WriteAsJsonAsync(response);
    }

    private static async Task HandleExceptionAsync(HttpContext context, Exception ex)
    {
        context.Response.StatusCode = StatusCodes.Status500InternalServerError;
        context.Response.ContentType = "application/json";
        var response = new { error = "Ocurrio un error interno. Intente mas tarde.", type = "ServerError" };
        await context.Response.WriteAsJsonAsync(response);
    }
}

public class JwtTokenService
{
    private readonly IConfiguration _config;

    public JwtTokenService(IConfiguration config)
    {
        _config = config;
    }

    public string GenerateToken(ApplicationUser user, IEnumerable<string> roles)
    {
        var claims = new List<Claim>
        {
            new Claim(JwtRegisteredClaimNames.Sub, user.Id.ToString()),
            new Claim(JwtRegisteredClaimNames.Email, user.Email!),
            new Claim(JwtRegisteredClaimNames.Jti, Guid.NewGuid().ToString()),
            new Claim("firstName", user.FirstName),
            new Claim("preferredCurrency", user.PreferredCurrency)
        };

        foreach (var role in roles)
            claims.Add(new Claim(ClaimTypes.Role, role));

        var key = new SymmetricSecurityKey(Encoding.UTF8.GetBytes(
            _config["JwtSettings:SecretKey"]!));
        var creds = new SigningCredentials(key, SecurityAlgorithms.HmacSha256);

        var token = new JwtSecurityToken(
            issuer: _config["JwtSettings:Issuer"],
            audience: _config["JwtSettings:Audience"],
            claims: claims,
            expires: DateTime.Now.AddMinutes(int.Parse(_config["JwtSettings:ExpiryMinutes"]!)),
            signingCredentials: creds);

        return new JwtSecurityTokenHandler().WriteToken(token);
    }

    public string GenerateRefreshToken()
    {
        var randomBytes = new byte[64];
        using (var rng = RandomNumberGenerator.Create())
            rng.GetBytes(randomBytes);
        return Convert.ToBase64String(randomBytes);
    }
}

public class AuthService
{
    private readonly HttpClient _httpClient;
    private readonly ILocalStorageService _localStorage;
    private readonly AuthenticationStateProvider _authStateProvider;

    public AuthService(IHttpClientFactory factory,
        ILocalStorageService localStorage,
        AuthenticationStateProvider authStateProvider)
    {
        _httpClient = factory.CreateClient("FinTrackApi");
        _localStorage = localStorage;
        _authStateProvider = authStateProvider;
    }

    public async Task<AuthResult> LoginAsync(string email, string password)
    {
        var response = await _httpClient.PostAsJsonAsync("api/auth/login",
            new { Email = email, Password = password });

        if (!response.IsSuccessStatusCode)
        {
            var error = await response.Content.ReadAsStringAsync();
            return AuthResult.Fail(error);
        }

        var result = await response.Content.ReadFromJsonAsync<LoginResponse>();
        await _localStorage.SetItemAsync("authToken", result!.Token);
        await _localStorage.SetItemAsync("refreshToken", result.RefreshToken);

        ((CustomAuthStateProvider)_authStateProvider).NotifyUserAuthentication(result.Token);

        return AuthResult.Ok();
    }

    public async Task LogoutAsync()
    {
        await _localStorage.RemoveItemAsync("authToken");

public class CustomAuthStateProvider : AuthenticationStateProvider
{
    private readonly ILocalStorageService _localStorage;
    private readonly HttpClient _httpClient;

    public CustomAuthStateProvider(ILocalStorageService localStorage,
        IHttpClientFactory factory)
    {
        _localStorage = localStorage;
        _httpClient = factory.CreateClient("FinTrackApi");
    }

    public override async Task<AuthenticationState> GetAuthenticationStateAsync()
    {
        var token = await _localStorage.GetItemAsync<string>("authToken");

        if (string.IsNullOrEmpty(token))
            return new AuthenticationState(new ClaimsPrincipal(new ClaimsIdentity()));

        var identity = new ClaimsIdentity(ParseClaimsFromJwt(token), "jwt");
        var user = new ClaimsPrincipal(identity);
        _httpClient.DefaultRequestHeaders.Authorization =
            new AuthenticationHeaderValue("Bearer", token);

        return new AuthenticationState(user);
    }

    public void NotifyUserAuthentication(string token)
    {
        var identity = new ClaimsIdentity(ParseClaimsFromJwt(token), "jwt");
        var user = new ClaimsPrincipal(identity);
        NotifyAuthenticationStateChanged(Task.FromResult(new AuthenticationState(user)));
    }

    public void NotifyUserLogout()
    {
        var anonymous = new ClaimsPrincipal(new ClaimsIdentity());
        _httpClient.DefaultRequestHeaders.Authorization = null;
        NotifyAuthenticationStateChanged(
            Task.FromResult(new AuthenticationState(anonymous)));
    }

    private static IEnumerable<Claim> ParseClaimsFromJwt(string jwt)
    {
        var payload = jwt.Split('.')[1];
        var jsonBytes = ParseBase64WithoutPadding(payload);
        var keyValuePairs = JsonSerializer.Deserialize<Dictionary<string, object>>(jsonBytes);
        return keyValuePairs!.Select(kvp => new Claim(kvp.Key, kvp.Value.ToString()!));
    }

    private static byte[] ParseBase64WithoutPadding(string base64)
    {
        switch (base64.Length % 4)
        {
            case 2: base64 += "=="; break;
            case 3: base64 += "="; break;
        }
        return Convert.FromBase64String(base64);
    }
}

public interface ICurrencyConversionService
{
    public Task<decimal> ConvertAsync(decimal amount, Currency from, Currency to);
    public Task<decimal> GetRateAsync(Currency from, Currency to);
    public Task<Money> NormalizeToUSDAsync(Money money);
    public Task<Money> NormalizeToVESAsync(Money money);
}

public class CurrencyConversionService : ICurrencyConversionService
{
    private readonly IRepository<ExchangeRate> _rateRepo;
    private readonly ILogger<CurrencyConversionService> _logger;

    public CurrencyConversionService(IRepository<ExchangeRate> rateRepo,
        ILogger<CurrencyConversionService> logger)
    {
        _rateRepo = rateRepo;
        _logger = logger;
    }

    public async Task<decimal> ConvertAsync(decimal amount, Currency from, Currency to)
    {
        if (from == to) return amount;

        var rate = await GetRateAsync(from, to);
        return amount * rate;
    }

    public async Task<decimal> GetRateAsync(Currency from, Currency to)
    {
        // Buscar tasa directa: USD -> VES
        var directRate = await _rateRepo.GetQueryable()
            .Where(r => r.FromCurrency == from && r.ToCurrency == to && r.IsActive)
            .OrderByDescending(r => r.Date)
            .FirstOrDefaultAsync();

        if (directRate != null)
            return directRate.Rate;

        // Buscar tasa inversa y calcular reciproco: VES -> USD
        var inverseRate = await _rateRepo.GetQueryable()
            .Where(r => r.FromCurrency == to && r.ToCurrency == from && r.IsActive)
            .OrderByDescending(r => r.Date)
            .FirstOrDefaultAsync();

        if (inverseRate != null && inverseRate.Rate != 0)
            return 1m / inverseRate.Rate;

        // Fallback: intentar via USD como moneda puente
        if (from != Currency.USD && to != Currency.USD)
        {
            var toUsd = await GetRateAsync(from, Currency.USD);
            var toTarget = await GetRateAsync(Currency.USD, to);
            return toUsd * toTarget;
        }

        _logger.LogWarning("No se encontro tasa de cambio de {From} a {To}", from, to);
        throw new DomainException(
            $"No se encontro tasa de cambio de {from} a {to}. " +
            "Por favor actualice las tasas de cambio.");
    }

    public async Task<Money> NormalizeToUSDAsync(Money money)
    {
        if (money.Currency == Currency.USD)
            return money;

        var converted = await ConvertAsync(money.Amount, money.Currency, Currency.USD);
        return new Money(converted, Currency.USD);
    }

    public async Task<Money> NormalizeToVESAsync(Money money)
    {
        if (money.Currency == Currency.VES)
            return money;

        var converted = await ConvertAsync(money.Amount, money.Currency, Currency.VES);
        return new Money(converted, Currency.VES);
    }
}

Application/Transactions/CreateTransactionHandler.cs
public class CreateTransactionHandler
    : IRequestHandler<CreateTransactionCommand, Result<Guid>>
{
    private readonly IRepository<Account> _accountRepo;
    private readonly ICurrencyConversionService _conversionService;
    private readonly IUnitOfWork _unitOfWork;

    public async Task<Result<Guid>> Handle(CreateTransactionCommand request, CancellationToken ct)
    {
        var account = await _accountRepo.GetByIdAsync(request.AccountId, ct);
        if (account == null)
            return Result<Guid>.Fail("Cuenta no encontrada");

        var amount = request.Amount;
        var currency = request.Currency;

        // Si la moneda de la transaccion difiere de la cuenta, convertir
        if (request.Currency != account.Currency)
        {
            try
            {
                amount = await _conversionService.ConvertAsync(
                    request.Amount, request.Currency, account.Currency);
                currency = account.Currency;
            }
            catch (DomainException ex)
            {
                return Result<Guid>.Fail(ex.Message);
            }
        }

        // Verificar fondos suficientes para gastos/transferencias
        if ((request.Type == TransactionType.Expense ||
             request.Type == TransactionType.Transfer) && amount > account.Balance)
        {
            return Result<Guid>.Fail(
                $"Fondos insuficientes. Balance: {account.Currency} {account.Balance:N2}");
        }

        var transaction = account.AddTransaction(
            request.Type, amount, currency, request.Description,
            request.Date, request.CategoryId, request.Tags, request.IsRecurring);

        await _unitOfWork.SaveChangesAsync(ct);

        return Result<Guid>.Ok(transaction.Id);
    }
}

Reportes con Normalizacion de Moneda

Application/Reports/Queries/GetExpenseReportQuery.cs
public class GetExpenseReportHandler
    : IRequestHandler<GetExpenseReportQuery, Result<ExpenseReportDto>>
{
    private readonly FinTrackDbContext _context;
    private readonly ICurrencyConversionService _conversion;

    public async Task<Result<ExpenseReportDto>> Handle(
        GetExpenseReportQuery request, CancellationToken ct)
    {
        var transactions = await _context.Transactions
            .Where(t => t.Type == TransactionType.Expense
                      && t.Date >= request.StartDate
                      && t.Date <= request.EndDate)
            .Include(t => t.Category)
            .ToListAsync(ct);

        // Normalizar todos los gastos a la moneda de reporte
        var normalizedExpenses = new List<NormalizedExpense>();
        foreach (var t in transactions)
        {
            var money = new Money(t.Amount, t.Currency);
            var normalized = request.ReportCurrency == Currency.USD
                ? await _conversion.NormalizeToUSDAsync(money)
                : await _conversion.NormalizeToVESAsync(money);

            normalizedExpenses.Add(new NormalizedExpense
            {
                Description = t.Description,
                Category = t.Category?.Name ?? "Sin categoria",
                OriginalAmount = t.Amount,
                OriginalCurrency = t.Currency.ToString(),
                NormalizedAmount = normalized.Amount,
                NormalizedCurrency = normalized.Currency.ToString(),
                Date = t.Date
            });
        }

        var groupedByCategory = normalizedExpenses
            .GroupBy(e => e.Category)
            .Select(g => new CategoryExpenseDto
            {
                CategoryName = g.Key,
                TotalAmount = g.Sum(e => e.NormalizedAmount),
                TransactionCount = g.Count(),
                Percentage = 0  // Calcular despues
            })
            .OrderByDescending(c => c.TotalAmount)
            .ToList();

        var total = groupedByCategory.Sum(c => c.TotalAmount);
        foreach (var cat in groupedByCategory)
            cat.Percentage = total > 0 ? (cat.TotalAmount / total) * 100 : 0;

        return Result<ExpenseReportDto>.Ok(new ExpenseReportDto
        {
            StartDate = request.StartDate,
            EndDate = request.EndDate,
            ReportCurrency = request.ReportCurrency.ToString(),
            TotalExpenses = total,
            CategoryBreakdown = groupedByCategory
        });
    }
}
